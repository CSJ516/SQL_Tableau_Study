[CHAPTER 09 실무에 필요한 Tableau]

9-1. 자주 쓰이는 Tableau 기능

(1) 퀵 테이블 계산 : 데이터 시각화에 필요한 계산식을 빠르게 적용할 수 있는 기능
** 누계/ 구성 비율/ 순위/ 전년 대비 성장률 등
(2) 누적 막대 차트 : 마크 카드 상단의 [막대] 클릭, [행]-[퀵 테이블 계산]-[누계]
** [행]-[선반에서 편집]을 클릭하여 누계(RUNNING_SUM) 함수 확인
(3) 월별 지표 (누계)
** 월별 테이블 생성 후 ~(카운트) 필드를 더블 클릭하면 측정값 이름, 측정값 선반 생성
   (~(카운트)는 해당 테이블의 행 건수를 반환하며, SQL COUNT와 동일한 개념)
** 계산하고자 하는 필드를 Ctrl 누른 상태에서 마크 카드(세부 정보)로 드래그
   (측정값 선반에 동일한 계산식을 복사할 수 없기 때문에 마크 카드에 복사)
** 드래그한 필드를 클릭하여 [퀵 테이블 계산]-[누계]로 변경
** 변경된 누계 필드를 측정값 선반에 드래그
** 테이블에서 누계 열을 우클릭하여 [별칭 편집]
** 측정값 선반에서 필요없는 필드는 빈 곳으로 드래그하여 제거

(4) 개별/ 단일/ 이중 축 : 여러 측정값의 축을 뷰에서 한 번에 보는 기능
(5) 개별 축 : 측정값 필드들을 개별 축으로 뷰에 표시할 때 사용
** 측정값 필드들을 행에 드래그하여 개별 축 생성
(6) 단일 축 : 측정값 필드들을 단일 축으로 뷰에 표시할 때 사용
** 행에 있는 필드 하나를 다른 하나의 그래프 축으로 드래그하여 단일 축 생성
** 행에 측정값 생성 및 측정값 선반 생성
(7) 이중 축 : 측정값 필드들을 이중 축으로 뷰에 표시할 때 사용
** 행에 있는 필드 하나를 클릭하여 [이중 축] 선택
** 그래프 축을 우클릭하여 [축 동기화]를 누르면 이중 축이 동일한 값으로 변경

(8) 기본 함수 : 숫자/ 문자/ 날짜/ 유형 변환/ 논리/ 집계/ 테이블 계산 총 7가지 자주 사용
(9) 숫자 함수 : ABS, MAX/ MIN, ZN, ROUND
** ZN(식) : null이면 0을 반환하고, 아니면 <식> 반환
(10) 문자 함수 : LEFT/ RIGHT, LEN, LOWER/ UPPER, REPLACE, CONTAINS, SPLIT
** SPLIT('a-b-c','-',2) = b : 구분 기호 뒤에 오는 조건과 반대
(11) 날짜 함수 : DATEADD, DATEDIFF, DATENAME, DATEPART, YEAR/ QUARTER/ MONTH/ WEEK/ DAY, TODAY, NOW, MAX/ MIN
** DATENAME('month', #2021-04-12#) = 'April' : 날짜 부분 기준으로 문자열 반환
** DATEPART('month', #2021-04-12#) = 4 : 날짜 부분 기준으로 숫자열 반환
(12) 유형 변환 함수 : DATE, DATETIME, FLOAT, INT, STR
(13) 논리 함수 : IF, IFNULL, ISNULL
** IFNULL(식1, 식2) : null이면 <식2>을 반환하고, 아니면 <식1>을 반환
** ISNULL : 올바른 데이터가 포함되지 않으면 true 반환
(14) 집계 함수 : ATTR, COUNT, COUNTD, SUM, AVG, MAX/ MIN, STDEV, VAR
** ATTR : 단일 값이 아니면 별표(*) 반환
** COUNTD : 고유 건수 반환 (SQL COUNT(DISTINCT)와 동일한 개념)
(15) 테이블 계산 함수 : INDEX, RANK, RANK_DENSE, RANK_MODIFIED, RANK_UNIQUE, RUNNING_AVG/ MAX/ MIN/ COUNT, TOTAL
** RANK (ex) 1위, 2위, 2위, 4위)
** RANK_DENSE (ex) 1위, 2위, 2위, 3위)
** RANK_MODIFIED (ex) 1위 3위, 3위, 4위)
** RANK_UNIQUE (ex) 1위, 2위, 3위, 4위)
