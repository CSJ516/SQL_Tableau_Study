[CHAPTER 10 실무에 자주 쓰이는 Tableau 데이터 시각화]

10-1. PPT 데이터 시각화를 Tableau로

(1) RFM 고객세분화 분석 : 고객들의 구매지표(최근성, 구매빈도, 구매금액) 등을 활용해 고객을 세분화할 때 사용
** 매출액 : [Price] * [Quantity]
** 구매금액 : { FIXED [Mem no], YEAR([Order Date]) : SUM([매출액]) }
** 구매빈도 : { FIXED [Mem no], YEAR([Order Date]) : COUNTD([Prod No]) }
   -> 연도별로 세분화된 회원 수 집계

(2) 지역별 매장 분석 : 매장 주소를 지리적으로 시각화할 때 효과적
** [Store Addr] 필드 클릭 후 [변환]-[사용자 지정 분할]
   -> 구분 기호 사용 : space(공백) / 분할 해제 : 2
** Shift 누른 상태에서 분할된 두 필드를 클릭 후 [계층]-[계층 만들기]
** 분할 필드1과 분할 필드2 앞 아이콘을 눌러 각각 시/도, 카운티로 지리적 역할 변경
** 앞서 생성한 계층을 마크 카드(세부 정보)로 드래그하여 지도 생성
** [위도] 필드를 복사하여 추가 생성된 축을 이중 축으로 뷰에 표시

(3) 카테고리 분석 : 파이 차트를 이중 축으로 시각화할 때 효과적
** 마크 카드 상단의 [파이 차트] 클릭
** 열 선반을 더블 클릭하여 MIN(1)을 2번 입력 -> 그래프 2개 생성
** 마크 카드 상단에 집계(MIN(1)) 하나를 수정
   -> [Brand] 및 합계(매출액) 필드 제거, 합계(매출액) 필드의 [테이블 계산 지우기] , 크기 축소
** 행에 있는 오른쪽 집계(MIN(1))를 이중 축으로 변경

(4) 워드 클라우드 분석 : 텍스트를 크기로 시각화하고자 할 때 사용
** 마크 카드 상단의 [텍스트] 클릭
** 매출액에 따른 모델명 시각화
   -> 마크 카드(색상 및 크기)에 [매출액] 필드 드래그
   -> 마크 카드(텍스트)에 [Model] 필드 드래그
