[CHAPTER 08 데이터 시각화에 필요한 Tableau]

8-1. 필터

(1) 데이터 원본 필터 : 연결된 원본 데이터에 필터 적용. 대용량 데이터를 불러올 때 유용
** Tableau 새 창에서 테이블을 결합한 후 필터 추가 및 편집
(2) 워크시트 페이지는 필터 이동 및 재생을 통해 워크시트 뷰를 동적으로 보이게 하는 기능
** 워크시트의 페이지 칸에 필드를 드래그하여 필터 추가 및 편집
** 우측 상단에 수동 페이지 이동, 자동 페이지 재생, 기록 표시 옵션 생성
(3) 워크시트 필터 : [차원] 및 [측정값] 필드를 통해 워크시트 뷰를 필터하는 기능
** 워크시트의 필터 칸에 필드를 Ctrl 누른 상태에서 드래그하여 필터 추가 및 편집
** 필드가 차원(분류적인 데이터)일 경우 일반/ 와일드카드/ 조건/ 상위 기능 생성
** 필드가 측정값(집계적인 데이터)일 경우 값 범위/ 최소/ 최대/ 특수 기능 생성

8-2. 필드 및 뷰

(1) 필드 변환 : [차원] 및 [측정값] 필드를 활용하여 시각화에 필요한 새로운 필드로 변환
** [차원] 및 [측정값] 필드를 클릭하면 만들기/ 변환/ 변경 등의 기능 표시
(2) 별칭 : [차원] 필드 값을 변환할 때 사용
** [차원] 필드 클릭 후 별칭 생성. 별칭은 반드시 고윳값으로 지정
(3) 계산된 필드 : 함수를 활용해 필드 계산
** 검색 창 옆 화살표를 클릭한 뒤 계산된 필드 만들기
** IF ~ THEN ~ ELSEIF ~ ELSE ~ END 구문 (함수는 CHAPTER 09 참고)
(4) 그룹 : [차원] 및 [측정값] 필드 값을 그룹화. 주로 [차원]을 그룹화할 때 사용
** 필드 클릭 후 [만들기]-[그룹]
(5) 집합 : [차원] 필드 값을 집합으로 나눌 때 사용
** 동적 집합 : 원본 데이터에 따라 집합이 변경되며, 단일 차원만 가능
   -> [차원] 필드 클릭 후 [만들기]-[집합]
   -> 행에 생성된 동적 집합 필드를 드래그하면 IN/OUT으로 구분
** 고정 집합 : 고정된 집합이며, 단일 or 다중 차원 가능
   -> 차트상 일부 데이터 드래그-[집합 만들기]
   -> 생성된 고정 집합 필드를 마크 색상으로 드래그하면 IN/OUT을 색상으로 구분
(6) 구간차원 : [측정값] 필드 값을 일정한 크기별로 나눌 때 사용. 새로운 [차원] 필드 생성
** [측정값] 필드 클릭 후 [만들기]-[구간차원]
(7) 매개 변수 : 사용자가 직접 변수를 활용하여 뷰를 필터할 때 사용
** 검색 창 옆 화살표를 클릭한 뒤 매개 변수 만들기
** 생성된 매개 변수 클릭 후 [매개 변수 표시]
** 계산된 필드 생성 시 매개 변수 활용 가능. Ctrl 누른 상태에서 필터 칸으로 드래그하면 필터로도 사용 가능
(8) 변환 : [차원] 필드 값을 여러 필드로 나눌 때 사용
** 분할 : 반복 패턴이 있을 경우 Tableau에서 자동 분할
   -> [차원] 필드 클릭 후 [변환]-[분할]
** 사용자 지정 분할 : 구분 기호로 수동 분할
   -> [차원] 필드 클릭 후 [변환]-[사용자 지정 분할]
(9) 계층 : [차원] 필드 값들을 하나의 계층으로 만들 때 사용
** Shift 누른 상태에서 계층으로 만들 [차원] 필드들 클릭 후 [계층]-[계층 만들기]
(10) 데이터 유형 및 지리적 역할 : [차원] 및 [측정값] 필드 값의 데이터 형식을 변환할 때 사용
** 데이터 유형 : 숫자/ 날짜/ 문자열 변환
** 지리적 역할 : 문자열 중 국가/ 시도/ 카운티(시군구) 등으로 변환
   -> [차원] 필드 앞 아이콘을 클릭한 뒤 [지리적 역할]
   -> 마크 색상 및 세부 정보에 드래그하면 행과 열에 위도 및 경도, 뷰에 지도 생성
   -> 마크에 드래그한 필드의 [+]를 클릭하면 계층 확인 가능
   (지도 위치 표시가 안 될 경우 [맵]-[위치 편집]-[국가/지역]을 대한민국으로 변경)
(11) 차원 및 측정값 변환 : 차원을 측정값 or 측정값을 차원으로 변환
** 필드 클릭 후 [측정값으로 변환] or [차원으로 변환]
   -> 측정값으로 변환할 때 카운트(고유)는 중복된 값을 제외하고 고유한 값 반환
(12) 워크시트 우클릭 후 [색상]을 클릭하면 워크시트 색상 변경 가능

(13) 뷰 편집 : 서식/ 축 편집/ 추세선/ 주석 등을 활용해 뷰를 시각적으로 편집
(14) 서식 : 글꼴/ 맞춤/ 음영/ 테두리/ 라인을 활용해 뷰 서식 변경
** 뷰에서 우클릭 후 [서식] -> 전체 및 특정 필드의 서식 변경 가능
(15) 축 편집 : 범위/ 눈금/ 축 제목/ 눈금선을 활용해 [측정값]들의 축 편집
** 축에서 우클릭 후 [축 편집]
(16) 추세선 : 선형/ 로그/ 지수/ 거듭제곱/ 다항식을 활용해 [측정값]들의 추세 표시
** 뷰에서 우클릭 후 [추세선]-[추세선 표시] -> 선형 추세선 자동 생성
   -> 선형 추세선 우클릭 후 [모든 추세선 편집]
(17) 주석 : 마크/ 지점/ 영역을 활용해 뷰에 주석 추가
** 차트상 데이터 우클릭 후 [주석 추가]-[마크 or 지점 or 영역]
   -> 생성된 주석 마크 더블 클릭으로 주석 편집, 우클릭으로 서식 편집

8-3. 마크 카드 및 표현 방식

(1) 마크 카드 : 색상/ 크기/ 레이블/ 세부 정보/ 도구 설명을 활용해 뷰를 시각적으로 표시
** 마크 카드 상단을 클릭하여 차트 유형 변경 가능 ex) 자동 -> 파이 차트
(2) 세부 정보에 필드를 드래그하면 차트상에서 세부 정보 확인 가능
(3) 도구 설명 클릭 후 [삽입]-[시트]-[마크 카드(색상,크기,레이블)] -> 차트상에서 시트 정보 확인 가능
(4) 표현 방식 : 선택한 필드 값들을 기준으로 적합한 뷰 추천
** Ctrl 누른 상태에서 여러 필드를 선택하면 오른쪽 상단의 표현 방식 자동 추천

8-4. 분석

(1) 요약 : 상수 및 평균 라인/ 사분위수 및 중앙값/ 박스 플롯/ 총계를 활용해 뷰를 분석적으로 요약
(2) 상수 및 평균 라인 : 측정값 필드의 상수 및 평균 요약정보가 뷰에 표시
** 사이드바 상단의 분석 탭을 클릭하여 상수 라인 or 평균 라인을 뷰에 드래그
(3) 사분위수 및 중앙값 : 측정값 필드의 사분위수 및 중앙값 요약정보가 뷰에 표시
** 사이드바 상단의 분석 탭을 클릭하여 사분위수 및 중앙값을 뷰에 드래그
(4) 테이블, 패널, 셀 : 분석할 항목의 범위 정의
** 연도별 브랜드 매출액 차트를 통해 살펴보면 테이블은 뷰에 있는 전체 데이터,
    패널은 연도별, 셀은 연도 및 브랜드별로 분석할 항목의 범위 정의
(5) 박스 플롯 : 측정값 필드의 사분위수 및 중앙값 외, 수염 요약정보를 박스 형태로 뷰에 표시
** 하안 및 상안 수염 : 1, 3사분위수에서 1.5배를 곱한 값 (이를 벗어나면 이상치)
** 산점도로 나타내기 위해 마크 카드 상단을 클릭하여 [원]으로 변경
   -> 마크 카드에 필드를 드래그한 후 차원 및 불연속형으로 변경
   -> 사이드바 상단의 분석 탭을 클릭하여 박스 플롯을 뷰에 드래그
(6) 총계 : 측정값 필드의 총계가 뷰에 표시
** 사이드바 상단의 분석 탭을 클릭하여 총계를 뷰에 드래그 (소계/ 열 총합계/ 행 총합계)
** 맨 상단의 [분석]-[총계]를 통해 표시/ 위치/ 계산 변경 가능
